## 기술 스택

- **Frontend**: React, TypeScript, TailwindCSS, Vite
- **Backend**: Express, Node.js, MongoDB
- **AI**: OpenAI GPT API
- **배포**: Vercel (프론트), Render (백엔드), MongoDB Atlas
- **기타**: JWT, Axios, React Query

## 주요 기능

- 🔐 사용자 회원가입 및 로그인 
- 🤖 GPT API를 통한 운동 루틴 생성
- ✏️ 운동 루틴 편집 및 저장
- 📊 운동 히스토리 조회
- 🗨️ 댓글 작성
